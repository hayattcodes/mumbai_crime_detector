# Mumbai Crime Area Detector
The Mumbai Crime Area Detector is a browser-based analytics dashboard providing ward-level crime data analysis across all 24 Brihanmumbai Municipal Corporation (BMC) wards within a unified, interactive interface. The platform tracks 10 distinct crime categories from 2020 to 2024 and utilizes Ordinary Least Squares (OLS) linear regression to generate predictive insights for 2025.Designed for citizens, researchers, urban planners, and policy analysts, the application is completely client-side and requires no local installation.Core FeaturesInteractive Spatial Mapping: Color-coded bubble map visualizing safety scores and crime volumes across all 24 wards, featuring granular drill-down capabilities via marker selection.Granular Ward Analytics: Dedicated detail panel rendering safety scores (0–100 range), percentage-based crime breakdown charts, mini-trend sparklines, and 2025 predictive ranges.Statistical Forecasting: 2025 crime projections calculated via OLS linear regression, complete with 95% confidence intervals displayed as intuitive lower and upper bounds.Crime Density Heatmap: A 24-ward by 10-crime-type matrix designed for instantaneous hotspot identification.Safety Leaderboards: Comparative ranking system displaying the safest and highest-risk wards alongside a synchronized city-wide average gauge.Macro City Overview: High-level analytics including zone-wise distributions, crime-type donut charts, and year-over-year trend lines.Multi-Modal Navigation: Synchronized ward selection through map interactions, drop-down search queries, or KPI cards.Technical StackLayerTechnologyVersion / SpecificationUser Interface & FrameworkStreamlit$\ge$ 1.31.0Data VisualizationPlotly5.18.0Data Engineeringpandas, NumPypandas $\ge$ 2.0.0, NumPy $\ge$ 1.24.0Statistical ModelingOLS Linear Regressionnumpy.polyfitGeospatial MappingPlotly go.ScattermapboxCarto Positron Basemap TilesDeployment TargetStreamlit CloudManaged ServiceProject StructurePlaintextmumbai-crime-detector/

Methodology: 2025 Predictive ModelingFor each municipal ward, the application fits an Ordinary Least Squares (OLS) linear regression model against five historical annual data points (2020–2024) using the numpy.polyfit framework.To compute the variance of the prediction, the platform calculates the full prediction standard error ($SE_{\text{pred}}$) using the following formula:$$SE_{\text{pred}} = SE_{\text{residual}} \times \sqrt{1 + \frac{1}{n} + \frac{(2025 - \bar{x})^2}{\sum (x_i - \bar{x})^2}}$$Where:$SE_{\text{residual}}$ is the residual standard error of the regression.$n$ is the number of data points ($n = 5$).$\bar{x}$ is the mean of the historical years.$x_i$ represents individual historical years.The resulting value is multiplied by a critical value of approximately 2 to establish the 95% confidence interval.
